## 수정필요

<br>

- Add Snippet → New Snippet
- 단축키 수정 : Ctrl + A → Ctrl + N
    - Ctrl + A(전체선택)이 쓰일 경우가 있음

<br>

- 정렬옵션 만들기
    - 복사(최신순)
    - 등록일자(최신순, 오래된순)
    - 수정일자(최신순, 오래된 순)
    - 제목 순(A → Z, Z → A)
    - 커스텀(위로 이동, 아래로 이동) → Detail에 버튼을 만들 수 있으면 함

<br>

- 설정 추가
    - 검색창이 비었을 때 정렬 옵션
        - 기본값 → 복사(최신순)
    - 검색창이 입력되었을 때 정렬 옵션
        - 기본값 : 제목순(A → Z)

<br>

- 모델 추가
    - 등록일자 : 모델에 없다면 가장 오래된 것으로 취급
    - 수정일자 : 모델에 없다면 가장 오래된 것으로 취급
    - 복사일자 : 모델에 없다면 그냥 없음

<br>

- 복사일자를 만드려면 Command 를 다시 만들어야 함
    - SnippetCopyTextCommand
    - SnippetOpenUrlCommand
