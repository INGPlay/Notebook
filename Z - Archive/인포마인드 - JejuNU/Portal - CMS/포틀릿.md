## 포틀릿

<br>

- 포틀릿 템플릿 관리
    - 포틀릿 생성

<br>

- 포틀릿 배치
    - 공통레이아웃
        - 웹 위쪽 포틀릿 영역
    - 기본레이아웃
        - 웹 아래쪽 포틀릿 영역
    - 모바일 레이아웃
        - 모바일 포틀릿 영역

<br>

- 포틀릿 타입
    - HTML 포틀릿
        - 기본 경로 : /WEB-INF/velocity/portlet/
        - 지정된 폴더 안에 있는 portlet.htm 호출
    - SQL 포틀릿
        - HTML 포틀릿에 SQL 호출
    - 대표홈페이지 포틀릿
        - 대표 홈페이지([https://cms.jejunu.ac.kr/api/contents/get.jsp](https://cms.jejunu.ac.kr/api/contents/get.jsp)) 에서 가져옴
        - 콘텐츠 칸의 숫자가 키 값이다.
        - https://cms.jejunu.ac.kr/api/contents/get.jsp?seq={콘텐츠}
            - 예시 - https://cms.jejunu.ac.kr/api/contents/get.jsp?seq=2434
