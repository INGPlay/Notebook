## 포틀릿 : 순환버스노선

<br>

- 노선은 관리자 페이지의 ‘코드관리’ 에서, 좌표는 파일로, 시간은 데이터베이스에서 관리하고 있음
- 노선
    - 환경설정 → 코드관리 → \[BUS\]버스노선
- 좌표
    - ```
FileUtil.getSaveDirectory("bus")+"/"+id
```
    - ```java
운영서버 파일위치 : /jnujeus/jnunext_files/portal_files/bus/A or B
```

- 시간표
    - ```
select PBM.* FROM POT_BUS_MST PBM
```
