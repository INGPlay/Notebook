## 메뉴 관련 테이블

<br>

- dev 접속

```java
SELECT * FROM CMS_MENU
```

<br>

- 사이트 별 조회

```java
SELECT * from CMS_MENU WHERE CM_SITE = 'ADMIN'
```

<br>

- 그룹 별 조회

```java
SELECT * from CMS_MENU WHERE CM_GROUPS = 'SADMIN'
```

<br>

<br>

\*CMS\_ADMIN 추가 쿼리

```java
UPDATE CMS_MENU 
SET CM_GROUPS = REPLACE(CM_GROUPS, 'SADMIN', 'CMS_ADMIN,SADMIN')
WHERE CM_SITE = 'ADMIN';
```
