## 언어관리

<br>

- 서버 시작 시 적용됨(서버 실행 중 수정은 적용안됨)

<br>

- 메뉴
    - 메뉴명에 “(ID)||(메뉴명)” 로 입력하면 언어관리에 등록됨
        - 시스템코드 : (ID)

<br>

- 포틀릿
    - 포틀릿 등록 시 언어관리에 코드가 등록됨
        - 시스템코드 : “menu\_” + (경로명)
    - <title>태그에 \_L.get(’(ID)||(메뉴명)’) 형태로 적어줘야 함
        - ```
<title>${_L.get('menu_Noncurricuar_Program||비교과프로그램')}</title>
```

<br>

- 수정 시에는 언어관리에서 수정이 안된다 → 직접 등록해줘야 함

<br>

- 관리자 메뉴의 타이틀에서 언어관리를 사용할 수 있게 수정했음

    - 웹 : <title> 태그가 사용됨
