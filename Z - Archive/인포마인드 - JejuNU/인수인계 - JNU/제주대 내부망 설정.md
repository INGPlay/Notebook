## 제주대 내부망 설정

<br>

- SMB 프로토콜 열기
    - 미 수행 시 ’지정된 네트워크 이름을 더 이상 사용할 수 없습니다.’ 에러 발생

```
제어판 -> windows 기능 켜기/끄기 -> SMB 1.0/CIFS 파일 공유 지원 -> 하위 폴더 모두 체크 후 확인 -> 재부팅
```

- 내부 네트워크 설정

```
시작 -> '이더넷 설정' 검색 -> 네트워크 프로필 유형 '개인 네트워크'로 설정 -> IP 할당 '편집' 버튼 선택 -> 할당된 IP 입력
```
