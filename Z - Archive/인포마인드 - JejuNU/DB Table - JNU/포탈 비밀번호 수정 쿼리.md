## 포탈 비밀번호 수정 쿼리

<br>

\*CS\_USER\_PWD\_M 테이블은 ‘AXIS’ 권한 필요

\*TIS\_EA\_USER\_PSWD\_HSTRY\_V 테이블은 ‘dev’, ‘dreamy’ 권한 필요

<br>

- 재학생
    - 2019106170
    - 2021102118
    - 2021215004

```java
UPDATE CS_USER_PWD_M
   SET USER_PWD = F_CS_ENCODE_HASH('salt123*')
WHERE USER_ID IN ('2019106170', '2021102118', '2021215004')
;
UPDATE TIS_EA_USER_PSWD_HSTRY_V
   SET LLSTDT = SYSDATE 
WHERE USER_ID IN ('2019106170', '2021102118', '2021215004')
;
```
