## 2025-03-26 인터뷰 정리 - 지꺼진 반란 

<br>

- 개요

1. 프로세스 중 여러 과 사이에 여러 공문이 전달되면서 생기는 비효율 개선
2. 지꺼진 반란 프로세스 확인 및 개선 논의
3. 오픈시기 지연

<br>

<br>

- 내용

1. 프로세스
    - 성적 경고자만 대상으로 하는 관리 기능 추가
2. 성적 경고자 상담
    - 상담 경고자 상담이 추가됨
    - 상담 경고자 상담은 교수상담 시기와 무관함
    - 성적 경고자 명단이 나오면 알아서 진행됨 (학생 신청 불필요)
    - 교수 상담(ex. 2025년 1학기) 시기와 성적 경고자 상담(ex. 2024년 2학기) 시기는 1학기 차이남
        - \*이해 못한 부분

3. 성적 경고자 조회
    - 성적 경고자 조회 → 비교과 교육센터에서 권한 필요
    - 성적 경고자 조회는 확정이 된 다음 보여야 하며, 각 과별로 확정 전의 성적 경고자가 보이는 메뉴 또한 필요
    - 학생은 성적 경고자 조회 기능 불필요
    - 메뉴 위치 : 학생경력관리 - 통합상담관리 메뉴 내 아래 쪽
4. 지꺼진 반란 대상자 관리
    - JNU 클래스에서 학생이 상담 종류(택1)를 선택한 이후, 시스템에서 이를 확인할 수 있는 상담 신청 현황을 확인할 수 있는 페이지 필요
    - 상담 신청 시 업로드 기능으로 지꺼진 반란 대상자를 확정 시킬 수 있는 기능 필요 (일정 관리 X)
        - 신청 시 당시의 학적 저장
    - 조회 권한과 업로드 권한을 각각 다르게 부여
5. 지꺼진 반란 (상담)결과 관리
    - 성적 경고자 전부를 대상으로 하는 지꺼진 반란(상담, 컨설팅) 진행상황 조회 페이지
        - 지꺼진 반란 현재 결과를 볼 수 있음
        - 상담 종결, 진행 중 등의 상태 표시
    - 지꺼진 반란(상담, 컨설팅) 진행 완료 확정 기능 필요
    - 성적 확정 기능 필요
        - 현재 성적이 확정되는 시기가 불명확하고, 성적 확정되더라도 계속 변경될 수 있음
    - 성적 미달 시, 미이수 처리
6. 오픈 일정
    - 2026년 2월 말 예정 (성적 경고자가 확정되는 시기)
