- 내용
    - 모집기준
        - 자유전공
            - 현재로서는 입학년도 기준 모집
        - 글로벌 자율전공
            - 학기별 모집

    - 학과변동 매핑 테이블
        - 자유전공
            - 지금까지 거의 1학기(3월)에만 학과가 바뀌었지만, 학기 단위로 학과가 바뀔 수 있는 가능성이 있음
            - 이름과 코드 전부 바뀜, 학과가 쪼개지거나 합쳐지는 경우를 대응해야 함
            - 기존학과 → 최종 대치학과만 담당자가 변경 (모든 변경 기록을 저장하지 않고)
                - 예시1) A → B 학과가 바뀐다면 테이블 예시
                    - |     |     |
| --- | --- |
| 기존학과 | 대치학과 |
| A   | B   |

                - 예시2) A → B → C 순서로 학과가 바뀐다면 테이블 예시
                    - |     |     |
| --- | --- |
| 기존학과 | 대치학과 |
| A   | C   |
| B   | C   |

<br>

            - A → B에서 A → B → C로 변경된다면 위 예시와 같이 담당자가 대치학과 컬럼을 전부 바꿔줘야 함

    - 희망전공신청
        - 1차 희망전공신청
            - 1학기 - 2월 진행
            - 1차 희망전공신청은 입학전이므로 수험번호로 받을 수 밖에 없다
                - → 수험번호로 조회할 수 있는 거를 만들어줘야 함

        - 2차 희망전공신청
            - 2학기 - 6월 진행
            - 2학기 희망전공신청 완료 시 반드시 전공신청을 해야 함. (2학년 1학기 휴학하더라도 전공 신청됨)
        - 희망전공신청은 3차 이후로도 계속 열려 있음
        - 자유전공 희망전공신청에 1, 2, 3 지망 적도록하기 → 포탈, 통합 변경
        - 한 학기에 한 번만 희망전공신청 할 수 있음 (신청기간 내에 수정은 가능)
    - 희망전공신청 통계
        - 학생들의 전공탐색 및 학사관리 차원으로 인해 필요함
        - 전공신청 직전의 희망전공신청 정보를 조회하는 게 필요한 듯
            - → 차수는 필요없고, 학기 별로 관리하기
            - → 재학, 휴학 상태 조회 추가
            - → 연도기준(신청연도, 입학연도)에 ‘전체’ 조건 추가

    - 전공신청
        - 2학기 - 신청 11월, 학적이관 3월
        - 1학기에 만약 한다면 - 신청 5월, 학적이관 8월
        - 승인 후 전공 변경 경우 → 없다고는 못함
        - 전과하면 전공신청은 하지 않음 (1학년도 전과 가능)
        - 학적 마스터 테이블의 학기 수를 신뢰할 수 없다
            - → 학기가 고정될 만한게 있는지 알아보기 (가학년, 가학기 정보 등등)

<br>

    - 페이지 별 요청사항
        - ‘연도’(문구) → 입학연도, 신청연도 처럼 기준을 명확히 알 수 있게끔 변경 요청 받음
        - 포탈 → 전공 선택 시, 콤보박스에 대학이름과 전공명 전부 보이도록 함
        - 포탈 - 자유전공신청 시 개인정보 동의 → 추후 문구 요청
        - 포탈 - 희망자유전공신청 시 경고 문구 → 추후 문구 요청
        - 통합 → 신청학과/전공 컬럼에 합쳐진 이름을 희망대학, 희망학과, 희망전공의 3개 컬럼으로 분리
        - 포탈, 통합 → 전공신청현황
            - 자유전공 - 1, 2, 3지망 순위, 전체 신청자 수 추가
            - 자율전공은 - 1지망만 (1, 2, 3지망 으로 변경 시 따로 요청)
        - 통합 - 중도탈락현황 - 대학알리미에 있는 자퇴현황을 보고 요청한 것
            - → 어떤 내용이 들어갈지 정리 요청
        - 통합 - 희망전공통계
            - → 어떤 내용이 들어갈지 정리 요청
        - 통합 - 전공신청학과승인 
            - 미승인될 시 학생이 뭘 해야 할지
                - 자유전공 → 학적이관만 안되도록 하기
                - 글로벌자율전공(글로벌자유전공승인, 학과승인, 본부승인 필요) → 나중에 어떻게 처리할지 내용 요청

        - 통합 - 자유전공대상자관리 - 학적이관 기능

                - 학사이관 담당자가 자유전공 페이지로 직접 들어와서 함
                - 권한은 주긴 할텐데 담당자가 페이지를 인지해야 함
                - 자유전공 - 교육혁신과에서 학적이관 함
                - 글로벌자율전공 - 담당 모름 → 담당부서나 담당자 내용 요청
