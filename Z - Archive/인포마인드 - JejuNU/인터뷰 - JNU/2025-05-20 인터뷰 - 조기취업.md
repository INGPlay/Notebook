- 내용
    - 조기취업 프로세스
        - 조기취업 학생들이 취업해서 학교를 못 올 때 대체 교과목
        - 안와도 교과목 교수가 과제를 부여해서 출석을 인정함
        - 신청서 작성하여 출석 대체자료 관리
    - 필요성
        - 조기취업 했더니 자료가 없어서 경고 받은 적이 있음
    - 매학기 초에 알림이 나감, 그 후 감사
    - 자료 보내기 - 소속학과, 교과목 교수 소속학과
    - 조기취업 취소되는 경우 - F학점이거나 그냥 안나가는 경우
    - 조기취업 관리 파일이 없음
    - 서류 - 신청서, 사대보험, 재직증명서
    - 학과에서 우리에게 도착하면 시작 → 한번에 처리
        - 근데 학과가 늦게 보냄
        - 시스템화하면서 지침 변경 가능성
    - 신청 기간이 있어야 할 거 같다
        - 일정 관리 → 기간을 짧게는 못잡을 거 같고…
        - 기간을 길게, 짧게 - 장단점 있음
        - 열 수 있는 기간(일정관리) - 본 신청기간, 이후 신청기간
            - 신청을 혁신과에서 관리할 수 있도록
            - 포털 신청
        - 출석인정신청서
            - 항목 - 그대로 갈 듯
            - 제출 서류 - 첨부파일
        - 금오공대 예
            - 신청서를 받지 않고 → 일단 신청했다가 학기 말에 서류를 올려나감
        - 수기 신청서는 받고 싶게 하고 싶다 → 교수가 받아야
        - 신청서를 받을 때 제출서류를 받는다
            - 첨부파일 → 수기 확인
        - 승인 처리 → 교수
            - 반려 가능
        - 학교를 다니다가 퇴사할 때
            - 승인 취소 화면?
        - 신청서만 넣어 놓고, 학기말에 서류를 올리게 하고 승인해야 함
        - → 증빙서류 신청할 때 받고, 학기말에 또 올리게끔하고 승인
        - 조기취업자 현황만 볼 수 있게하고, 
        - 승인 책임 - 교과목 교수, 학과, 교육혁신과 (지도교수는 들어갈 필요는 없을 듯)
            - 알림 줘야 함
            - 공동이냐 어느 한 곳만 승인되면 처리인가
                - 알아본다고 함, 정해지지 않음

        - 현재는 원격수업만 대상, 실기 수업은 이후에
        - 담당교수 없는 과목 → 담당자가 여러명이 될 수 있음 (관리부서 담당자)
            - 강사가 매번 달라지는 과목
            - 정하고 나서 얘기하기
        - 교수 소속학과 를 넣어야 되는지
            - 교수 소속학과까지는 과하다
        - 기존에 있던 서류 데이터를 넣고 싶다 요청
            - 입력 페이지 필요
        - 조기, 최종 승인 → 교육혁신과
            - 신청서도 같이 업로드
        - 현황 → 교수, 학과
        - 신청서, 첨부물 올라오면 알람
        - 금오공대 예시로 갈지, 새로나온 예시로 갈지 논의한다고 함
        - 4학기 1학기생들 대상
        - 연동되는 화면이 있는지?
