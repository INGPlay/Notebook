## \[정리\] 2025-05-26 인터뷰 - 자격시험

<br>

- 요청사항
    - 부실학회지 체크리스트 점검 완료 확인서
        - 항목이 자주변경되는지 확인 요청
    - 논문심사위원관리 - 석사랑 박사랑 입력하는 페이지를 분리
        - → 요청 측에서 명확히 어떻게 분리할 지 내용 요청
    - 자격시험합격기준관리
        - → 등록할 합격기준 요청
    - 신청서류 의견취합
        - → 6월 중순까지 하도록 요청

<br>

- 확인사항
    - 학위논문 심사청구서
        - 연구실적 테이블에서 가져와서 입력할 수 있는지 확인
    - 이수확인서내역을 끌고 올 수 있는지 확인

<br>

- 내용
    - 학위논문 심사청구서
        - 연구실적 테이블에서 가져와서 입력하기
            - → 할 수 있는지 확인
        - 첨부파일 : 성적증명서, 연구윤리교육이수확인서, 무통장입금사본
            - 성적증명서 - 수료생만 제출, 재학생은 직접 제출
            - 연구윤리교육이수확인서 - 수료한 학생들만 받도록 함
                - → 이수확인서내역이 있으면 끌고 올 수 있는지 확인

    - 부실학회지 체크리스트 점검 완료 확인서
        - 항목이 자주 변경되는지 → 확인해봐야 함
    - 논문심사위원관리 - 석사랑 박사랑 입력하는 페이지를 분리
        - → 요청 측에서 명확히 어떻게 할지 먼저 확인
    - 자격시험합격기준관리 - 합격기준 등록하기
        - → 합격기준 요청
    - 논문심사결과관리
        - → 논문 합격여부를 자동으로 끌어올 수 있게 하기
    - 논문심사외부위원관리 상세
        - 통장사본 첨부용 첨부파일 하나 추가
        - 심사위원 직급 가져오기
        - 외국인번호나 주민등록번호 없이 등록하기
            - → 주민등록번호 없애버리기

    - 기존논문심사
        - 변경없음
    - 디지털정보처 요청
        - 경력관리
            - 자격증 등록 시 이클립스로 트리거 쏴주고 있음
            - 뭔가 매핑되는 테이블이 없어서 뷰가 안나온다고 함

    - 외국어시험대체신청(학생)
        - 어학성적 끌어오기
        - 사인 - 승인관리 해야 되서 번거로울 수 있음
    - 외국어시험대체신청(학생)
        - 포탈 - 신청 페이지
        - 통합 - 조회 페이지
    - 외국어/종합시험최종결과관리
        - 석사 3과목, 박사 4과목 합격 시, 종합시험 합격날짜에 마지막 합격일자 적용
    - 신청서류 의견취합 → 6월 중순까지
    - 통합시스템에서 페이지를 확인할 수 있도록 권한 부여 요청
