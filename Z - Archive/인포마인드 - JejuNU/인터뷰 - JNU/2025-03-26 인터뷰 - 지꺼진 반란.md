## 2025-03-26 인터뷰 - 지꺼진 반란

<br>

- 참여자
    - 교육혁신과 양희정
    - 미래교육과 양희주

<br>

- 개요
    - 오픈시기를 늦추기 위함

<br>

- 내용

1. 변경될 프로세스
    - 성적 경고자만 대상으로 하는 관리 기능
    - 비교과와 지꺼진 반란은 따로 관리함
    - 상담 경로가 하나 더 생김 (성적 경고자 상담)
    - 성적 경고자 상담은 교수상담 시기와 무관함
    - 성적 경고자 명단이 나오면 알아서 진행됨 (학생이 신청할 필요 없음)
    - 교수 상담(ex. 2025 1학기)과 성적 경고자 상담(ex. 2024 2학기)은 1학기 차이 남
2. 평생 책임지도 
    - 성적경고자 명단
    - (공문) 왔다갔다하면서 벌어지는 시간 부족
    - 평생 책임 지도 교수 바뀌는 경우 → 애매한 기간?
    - 평생 책임 지도 교수 배정
3. 성적경고자 조회
    - 성적경고자 조회 → 비교과 교육센터 권한 필요
    - 성적 경고자 조회는 확정이 된 다음 보여야 한다
    - 각 과별로 확정 전의 성적 경고자가 보이는 메뉴 필요
    - 학생이 성적경고자 조회하는 거 → 포탈에서는 없어도 될 것 같다, + 신청
    - 메뉴 위치 → 학생경력관리 - 통합상담관리 밑쪽
    - 성적경고자 상담 이후 지꺼진 반란
4. 지꺼진 반란 프로세스
    - JNU 클래스에서 학생들이 상담 종류 선택 (택1)
    - 시스템에 학생들의 상담 신청 현황 확인 페이지 필요
    - 대상자 관리(상담결과 관리 비슷)
        - 상담 신청 시 대리 신청이 아닌 업로드 기능(확정), 지꺼진 반란 대상자 표시(일정은 나중에 잡아도 됨)
            - 신청 시 당시의 학적 저장
        - 조회권한, 업로드 권한 다르게 주기
    - 상담결과 관리(대상자 관리 비슷)
        - 지꺼진 반란 진행상황(컨설팅)이 보여지는 페이지 (성적 경고자 전부 대상)
        - 지꺼진 반란 결과를 비교과프로그램결과관리에서(비슷하게)?, 지꺼진 반란 결과(O, X)를 볼 수 있는 페이지
            - 지꺼진 반란 진행 완료 확정
            - 성적 확정 확정 : 언제 확정되는지 불명확해서 계속 확인 해야 함. 성적 확정 됐는지 보여줌.
                - 기존 성적 확정이 계속 변경될 수 있어서 불확실함

        - 상담 종결 진행 중 표시
        - 성적 미달 시, 미이수 있음
