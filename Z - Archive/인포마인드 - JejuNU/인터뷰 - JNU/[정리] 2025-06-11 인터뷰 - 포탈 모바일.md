## \[정리\] 2025-06-11 인터뷰 - 포탈 모바일

<br>

- 주제
    - 알람 배지, 자동로그인 처리에 대해 어떻게 할 것인가?
- 알람 배지
    - 현재 안드로이드는 되고, 아이폰은 안되는 상태
    - 안드로이드 배지 처리와 아이폰 배지 처리 방식이 다르다
    - 푸시서버에서 카운트 처리를 안하고 있어서 아이폰에서 처리가 안된다
    - 카운트 처리를 하면 아이폰에서 배지 처리가 될 텐데, 건건히 처리해야 할지, 통째로 처리해야 할지
    - 안드로이드처럼 통째로 처리하기
- 노티
    - 현재 노티 클릭 시 앱의 메인페이지로 감
    - 일단 노티 클릭 시 메시지 페이지로 가는 것만 해보기, 각각 페이지로 가는 건 처리할 게 많을 것 같다
- 알림 카운트 동기화
    - 아이폰에서만 고민, 안드로이드는 되고 있음
- 알림 전체삭제, 개별삭제, 보관 기능(☆ 아이콘)
    - 시안 필요
- 알림 삭제 배치
    - 6개월마다 삭제?
    - 알림 배치 처리를 장봉수 선생님 설계하도록 함
- 자동로그인
    - 아이폰에서만 문제가 생긴다?
    - 리로딩을 하는 이유?
    - SSO 끊겼을 때 재로그인 해결하면 해결이 된다?
    - 인트로 페이지에서 처리하기
- 기타
    - 아이폰에서 커서가 안나온다
    - 로딩 인디케이터가 없어서 느리면 진행되는지 알 수 없다
    - 큰 파일이 있어서 페이지 로딩이 너무 느린 것
    - 1순위 압축, 2순위 캐싱 처리하기
