- 내용
    - 요청사항 - 기준점수 변경되는 거, 포인트 누적? 학기별 초기화?
        - 기준점수는 학기 중에는 변경 안함
        - 누적은 전체 누적 (1~4학년), 신청해서 선발되면 까임 아니면 누적
    - 기준점수 변경
        - 예) 1학기 10점, 2학기 11점
        - 누적된 점수는 변경되지 않는가?
            - 2025학년도 내 에서는 바뀔 수 있다
            - 학기 내에서는 변동 불가, 학년(1년) 내에서는 변경 가능
            - 점수를 하향할 경우는 없다
        - 지침(기준점수)이 한 학기마다 나오나?
            - ?
        - 비교과 지침
            - 많이 등록되어 있지 않아가지고 진로기초지원, 취업지원 이 두 개만 가져온 거
            - 진로기초지원 - 빌드업, 취업지원 - 스킬업 으로 가고
            - 수료한 학생에 관해서만 점수
        - 포인트 부여시점
            - 비교과 같은 경우 수료했을 때 포인트 부여
            - 외국어 자격증 같은 경우 담당자가 승인을 해줬을 때 부여
            - 현장실습에서 승인 처리했을 때 부여?
            - 일경험 직업교육
                - 학점주는 거는 안주기로 해서 현장실습은 빠짐
                - 비교과 프로그램은 마일리지 이중 혹은 중복 신청 가능성 → 조금 더 고민
                    - 비교과 프로그램에서 자동으로 가져오는 거를 학생이 인지하기 어려움
                    - 중복을 막기 힘듦
                    - 입력 시 안내문구를 자세히 준다거나 하는 게 차선
                    - 장학금 주기 전에 한 번 확인
                    - 양 쪽에 다 주고 장학금 주기 전에 한 번 확인
                    - 등록은 가능해야 함
                        - 조금 더 고민

                - 일경험은 빠질 수도 있음
            - 사회봉사
                - 학점으로 걸러낼 수 있나? → 안됨
                - 시간 구간에 따라 점수 부여 → 가능
            - 졸업생과 지역청년 기준
                - 2년 이내 졸업생 → 졸업생, 넘으면 지역청년
                - 증빙자료(신분증 통장 사본) 관리 할 거임
                - 청년 제한은 추후 공지하면은 상관 없을 것 같다
                - 재학생에서 졸업생, 지역청년으로 바꼈을 때 점수 초기화할 것인지?
                    - 아이디가 다르면 통합 불가
                    - 다른 아이디에 포인트를 각각 쌓으면?
                    - 증빙할 때 신분증 받으니까
                    - 졸업하면 점수 0점으로 차감
                        - 로그가 쌓여서 되돌릴 수 있게 끔

                - 학기 중에는 안바뀐다. 년이 바뀌면 점수 고정된다
                    - 학기의 의미가 없어진다
                - 매년 초, 학기 초 → 전체 점수가 바뀐다?
                - 메뉴를 볼 수 있는 사본 → 장봉수 쌤에게 받기
                - 예상 마일리지 보여주다가, 마감되면 보여주기?
                - 학기 초에 정해서 1년 안으로는 안 바뀜 → 확정
                - 일부 점수를 빼고 싶을 때
                    - 차감 기준 정하기
                        - 항목 별로 라도 일부 체크해서 
                            - 관리자가 얼마나 차감할지 정해도 됨 → 미래교육과(비교과 마일리지)는 이렇게 함

                    - 외국어 자격증 중복 취득 시 실제 취득한 건 50점이지만 합쳐져서 90점처럼 보일 것 같다
                        - 높은 점수를 가진 자격만 채택이 되도록 하기?
                            - 이전 점수와 새로운 높은 점수 간의 차액 점수 반영?

                    - 차감기준 → 고민 → 추후 요청

            - 데이터 이관
                - 오픈할 때 이미 자격증 등록한 사람에게도 마일리지를 줄 건지?
                - 어학 자격 기간 → 입력을 안해서...
                - 대학입학 이후에 딴 자격증, 수상실적, 등등 만 인정
                    - 편입 → 편입 이후 부터 딴 것만 인정
                    - 자격증 취득일자와 학번 비교
                    - 봉사시작일자, 종료일자? → 종료일자가 학번 이후다 하면 넣는 걸로?
                    - 졸업생 → 졸업 이후 부터
                    - 지역청년 → 가입일자 이후 올린 거
                - 2025년 3월 데이터부터 반영
                    - 작년 마일리지는 반영이 안된다
                - 데이터 이관을 하는데, 취득일자를 가져오는거는 가능
