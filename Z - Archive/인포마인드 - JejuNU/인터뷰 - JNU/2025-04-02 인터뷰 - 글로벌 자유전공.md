- 내용

1. 자유전공 프로세스 확인
    - 교육혁신과의 신청안내로 프로세스 시작
    - 학생들이 희망전공, 전공배정, 자유전공 신청, 신청현황 확인, 학적이전 처리 할 수 있도록 하기
    - 각 학과에서 처리 현황을 알 수 있도록 하기
2. 학과 기준
    - 기준에 대한 입력 기준이 없음 
    - 입력 권한은 아마 교육혁신과에서 가져가야 한다. (확신하지 않음)
3. 자유전공을 신청하지 못했을 경우 최종 판단
    - 임의배정은 안된다고 함. → 어떻게든 만나서 처리함.
    - 자동배정 없음 → 수동으로 시스템에서 입력할 수 있도록 함.
4. 글로벌 자유전공과 자유전공의 차이
    - 외국인 대상인지 아닌지 (외국인 대상 → 글로벌 자유전공)
    - 입시전형 다름
    - 글로벌 자유전공에는 교육 과정이 있음
5. 학적이관 승인
    - 자유전공
        - 자유전공 → 본부
        - 각 학과 승인 단계에서는 인원까지만 조회 (최소정보 조회를 원함)

    - 글로벌 자유전공
        - 자유전공 → 지망학과 → 본부

6. 희망 자유전공 신청
    - 신청사유 X
    - 개인정보 (소속, 학번, 연락처, 이름)
    - 대학(인문대, 사회과학대), 학과, 날짜 
    - 동의버튼 (문구 넣고 동의, 비동의)
        - 문구 요청
    - 못 가는 학과를 선택하지 못하게 하기
7. 배정 신청
    - 11월까지는 완료되야 함.
    - 그 전까지 이런저런 준비로, 9월 말 오픈 예정
8. 학과별 기준 세팅 (글로벌 자유전공)
    - 학과별 기준 조사 후 어떤 기준이 필요한지 알려줄 예정
9. 교수상담 진행 횟수 조회 → 필요 없음
10. 자유전공 중도탈락률(이탈률) 인원현황
    - 자유전공 중도탈락률 : 자유전공 학생의 자퇴율
    - 통계 양식 요청 함
11. 학적부에서 자유전공과 일반학과의 전공 배정이 구분
    - 자유전공 학생들만 추적할 수 있다면 어떤 방식이든 상관 없음
12. 전공배정과 전과 동시 신청 시 (자유전공)
    - 학적 변동이력 어떻게 기록할지 → 추후 알림 예정
        - 둘 다 기록?, 하나만 기록?
        - 전과 우선?, 전공 배정 차선?

13. 휴학생 복학 시
    - 변동이력 받음
    - 자유전공
        - 1학기 2번을 했어도 전공배정은 가능하다 (학기마다 열려야 함, 연도 학기 기준)
        - 입학학과 선택 기준 년도 : 입학년도 기준
    - 글로벌 
        - 매학기 전공배정
        - 입학학과 기준 년도 - 입학년도 기준
