- 내용
    - 기준점수 변경되는 기준이 포인트 누적인가? 학기별 초기화 되나?
        - 점수누적은 1~4학년 전체 누적됨. 장학금 신청해서 선발되면 점수가 줄어들고 아니면 계속 누적됨.
        - 처음 기준점수를 각 학기 초에 정해서 학기 내에서는 변동 불가, 학년(1년) 내에서는 변경 가능 라는 입장이었다가, 처음 학기 초에 정해서 학년도 내에서 변경 불가 라고 함
            - 항목 별로 점수를 변동할 수 있는 기능은 만들어야 함

    - 지침(기준점수)이 한 학기마다 나오나?
        - 비교과 지침
            - 많이 등록되어 있지 않아 가지고 진로기초지원, 취업지원 이 두 개만 가져왔다고 함
            - 진로기초지원은 빌드업, 취업지원은 스킬업 으로 감

    - 포인트 부여시점이 어떻게 되나?
        - 비교과 프로그램 같은 경우는 수료했을 때 부여함
        - 외국어 자격증 같은 경우는 담당자가 승인 시 부여함
    - 일 경험 직업교육 관련
        - 현장실습에서 비교과 프로그램을 등록했을 때 자동으로 가져와서 부여할 시 다음과 같은 문제가 있어 좀 더 고민해본다고 함.
            - 비교과 프로그램에서 자동으로 가져와서 성장 포인트가 부여된다는 것을 학생이 인지하기 어려움
            - 성장 포인트 등록과의 중복 등록을 막기 어려움→ 이를 막기 위한 차선으로 성장포인트 등록 시 안내문구를 준다거나, 장학금을 주기 전에 관리자가 중복확인을 한다는 의견이 있었음 
        - 일경험 직업교육은 빠질 수도 있음 (가능성)
        - 사회봉사 관련 질문
            - 학점으로 걸러낼 수 있는가? → 안됨
            - 시간 구간에 따라 점수 부여가 가능한가? → 가능하다.

    - 졸업생과 지역청년 기준
        - 졸업 후 2년 이내인 졸업생, 아니라면 지역청년
        - 증빙자료(신분증, 통장 사본) 받아서 관리한다고 함
        - 청년 나이 제한은 법적으로 딱 정해진 게 아니라서, 장학금 공지 시 청년 나이 상한을 제시해야 함
        - 재학생에서 졸업생, 지역청년으로 바꼈을 때 점수 초기화할 것인지?→ 졸업하면 점수 0점으로 차감되며, 클라이언트는 로그가 쌓여서 차감 이전으로 되돌릴 수 있기를 원함
    - 일부 점수만 차감하고 싶은 경우 어떻게 할 것인가?
        - 하위 항목과 같은 제안이 있었음
            - 학생이 항목 별로 차감하고 싶은 것만 체크함
            - 관리자가 얼마나 차감할지 정함 → 미래교육과는 이렇게 한다고 함
        - 어학시험 관련
            - 외국어 자격증 중복으로 40점, 50점 이렇게 취득해서 올린다면, 실제로 취득한 거는 50점이지만 학생이 생각하기에 90점 처럼 보일 것 같다.
        - 결론
            - 고민해본다고 함

    - 오픈할 때 이미 자격증 등록한 사람에게도 마일리지를 줄 것인가?
        - 2025년 3월 데이터부터 반영 → 작년 마일리지는 반영이 안됨
        - 대학 입학 이후에 딴 자격증, 수상 실적 등등의 모든 자격만 인정됨
            - 자격증의 경우 취득일자와 학번을 비교해서 기준을 세움
            - 봉사활동의 경우, 봉사종료일자가 입학 이후인 것만 인정
        - 편입생에 경우, 편입 이후부터 딴 것만 인정됨
        - 졸업생에 경우, 졸업 이후부터 딴 것만 인정됨
        - 지역청년에 경우, 가입일자 이후로 올린 것만 인정됨
        - 데이터 이관 관련 질문
            - 어학 자격 기간을 가져올 수 있는가?→ 입력을 안해서 가져올 수 없음
            - 자격증 취득일자를 가져올 수 있는가?→ 가져올 수 있음
