## 2025-05-26 인터뷰 - 자격시험

<br>

- 개요
    - 요구사항 확인

<br>

- 내용
    - 논문신청
        - 학위논문 심사청구서
            - 이전에 연구실적 캡처해서 올렸던 거를 연구실적을 끌어들여서 입력 받게끔
                - 할 수 있는지 확인
            - 첨부
                - 1(성적증명서), 4(연구윤리교육 이수확인서), 6(무통장입금 사본) → 첨부파일
                    - 1 - 수료생만 제출, 재학생은 가져와서
                    - 4 - 수료한 학생들만 받도록하기 → 생각해보기
                        - 이수확인내역이 있으면 끌고 올 수 있는지 확인해보기
                    - ?? 요구사항 놓침
                - 7, 8 → 학과 제출
                - 공통관리 → 공통업무 → 조직코드관리 → 조직계열관리 : 계열에 따라 제출물이 다름

        - 학위논문 연구윤리 신청
            - 체크 하기
        - 부실학회지 체크리스트 점검 완료 확인서
            - 체크
            - 항목이 자주 변경되는지 → 확인해봐야 함

    - 논문심사위원관리
        - 석사랑 박사랑 입력하는 페이지를 다르게 할수 있는지 요청
            - 요청 측에서 명확히 어떻게 할지 먼저 확인
        - 점수관리하는 메뉴, 심사위촉하는 페이지로 분리
            - 안해도 된다고 함
        - ?? 요구사항 놓친거 같음
        - 자격시험합격기준관리 - 합격기준 받으면 등록 → 합격기준 요청
        - 논문심사결과관리 - 논문 합격여부를 자동으로 끌어올 수 있도록
        - 논문심사외부위원관리
            - 상세
                - 첨부파일 하나
                - 비고란 하나
                - 외국인번호나 주민등록번호 없이 등록하기 → 주민등록번호 없애버리기

        - 기존논문심사
            - 그대로
        - 디지털정보처 요청
            - 경력관리 → 자격증 등록 시 이클립스로 트리거 쏴주고 있음 → 뭔가 매핑되는 테이블이 없어서 뷰가 안나온다고 함
            - 자격시험 → 외국어/종합시험신청
                - 상관없는 페이지가 합쳐져 있음 → 한 메뉴에 있는게 낫다

        - 외국어 시험 대체신청(학생)
            - 어학성적 사본 첨부 → 어학성적 끌어오기
            - 입력을 한 곳으로 줄이기 → 그대로 놔두기
            - 사인 → 승인관리 해야 되서 번거로울 수 있음
        - 보고서 사항 승인관리 → 넣을지 말지 고민
    - 외국어시험대체신청(학생)
        - 포탈 신청, 통합 목록
    - 외국어/종합시험최종결과관리
        - 석사 3과목, 박사 4과목 합격 시, 종합시험 합격날짜에 마지막 합격일자 적용
    - 신청서류 의견취합 → 6월 중순까지
    - 통합시스템에서 페이지를 확인할 수 있도록 권한 부여 요청
