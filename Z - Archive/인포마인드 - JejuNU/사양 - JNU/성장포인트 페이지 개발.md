## 성장포인트 페이지 개발

<br>

- 통합
    - 성장포인트업무일정관리
        - 이번연도 3월 ~ 다음연도 2월 일정관리에 넣기
    - 성장포인트기준배점관리
        - 항목기준팝업 - 항목구분 ‘비교과 관리’ 추가
    - 적립현황
        - 상세보기 - 연도 검색조건 추가, 엑셀 버튼 빼기
        - 장학금신청버튼 추가 - 그리드에서 체크한 거
    - 장학금신청관리
        - 상세보기 - 연도 검색조건 추가, 엑셀 버튼 빼기, 경력선발구분 콤보박스 빼기
        - 경력선발구분 콤보박스 그리드에 추가
        - 선정된 학생에게 제출서류 팝업 버튼 추가
            - 은행구분, 계좌번호, 신분증, 통장사본

    - 메뉴추가 - 학생별 장학금 관리
        - 학생, 연도(전체) 검색 조건을 가진 조회 페이지
        - 상세보기 필요 없음
    - 점수추가 → 다시 논의
        - 항목기준관리의 선발기준 - 기타로 부여됨

<br>

    - \*상담, 비교과 타입의 경우 상세페이지 띄우지 말기
    - \*프로그램 설계 - 프로그램 목록 → 참조

<br>

- 포털
    - 적립내역
        - 신청하기 시 팝업(안내사항 같은 거 + 신청버튼)
    - 신청내역
        - 제출서류 팝업 버튼 추가
            - 은행구분, 계좌번호, 신분증, 통장사본
