## 제주 마라톤 - 결제 절차

<br>

- 통장 입금
    - DB에 참가자를 '입금 대기' 상태로  기록만 함

<br>

- 카드 결제
    - 요청 창 띄우기
        - [결제위젯 JavaScript SDK | 토스페이먼츠 개발자센터 (tosspayments.com)](https://docs.tosspayments.com/reference/widget-sdk#requestpayment%EA%B2%B0%EC%A0%9C-%EC%A0%95%EB%B3%B4)
    - 요청창 이후 리다이렉션하고 승인을 안함...
    - 결제 승인이 있어야 결제 취소 가능

<br>

<br>

- API 관련
    - POST /v1/payments/{paymentId}
        - RequestBody
            - orderId
            - amount
        - 의견
            - TossPay Docs나 다른 검색에는 안나오지만, confirm하는 api로 추정됨

<br>

- 버그
    - 결제창을 껐다가 다시 켜서 결제하면 화면이 안넘어감
        - ajax에 submit 사용되서 뭔가 이벤트 꼬인 거 같은데 잘 모르겠음
        - 일단 ajax 이벤트에서 submit을 뺌
