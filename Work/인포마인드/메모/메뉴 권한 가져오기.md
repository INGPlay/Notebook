## 메뉴 권한 가져오기

<br>

- page.tsx
    - 메뉴 버튼 권한 가져와야 됨
- actions.ts
    - 지금 호출한 action이 권한에 충족하는지 가져와야 함

<br>

- 키 후보
    - MENU\_CD
        - 여러 메뉴에서 향하는 경우, Menu 마다 적어줘야 함
    - PRGRM\_PATH
        - page.tsx의 경우 \`usePathname()\` 쓰면 됨
        - actions.ts는 `process.cwd()` 이거 참고해봐야 할듯
        - 여러 메뉴에서 향하는 경우에도 하나만 쓰면 됨

<br>

- 근데 page.tsx에서 actions.ts를 여러 개 바라보는 경우도 있을 수 있음

<br>

- 결국은 메뉴 버튼을 보여주는 거하고, 실질적인 API 하고 다를 수 밖에 없다

<br>

- 비로그인 호출은 Next.js에서 막을 수 있지 않을까?
