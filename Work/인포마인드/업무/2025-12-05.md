- 자격시험 확인요청

1. 연구윤리대체인증 컬럼 확인 (현재 사용하지 않는다고 함) - 이번에 추가된 컬럼 확인
2. (학과마다) 메뉴에서 (외부위원) 같은 사람을 여러 곳에 쓰고 싶은데 안불러와진다고 함. 확인해볼 것 - 여러곳에서 사용가능
3. 외국어시험통과여부, 종합시험통과여부, 연구윤리이수여부가 신청조건에 어떤 영향을 미치는지 확인
4. 포탈 학위청구논문신청에서 "박사학위논문 공표실적" 뭔지 추적해보기

<br>

- 재학생과 수료생
    - 공통
        - 대학원의 논문제출기한 체크
            - AJG\_THSS\_SBMSN\_TERM.FNSH\_AFTR\_YCNT 가 존재해야 함
        - 학위청구논문신청 요건 체크
            - 학위 청구 논문 신청의 논문합격여부(AJG\_DEGR\_CLM\_THSS\_APLY.THSS\_PASS\_YN)가 있어야 함 (이거 포탈에서 체크를 안하고 있었음)
            - 종합시험통과여부(FRGNL\_EXAM\_PASG\_YN)가 있어야 함

    - 수료생
        - 수료생 연구 에서 관리자 승인일 경우, 논문제출기한 상관없이 제출 가능
            - DREAMY.AHJ\_FNSH\_RSCR (수료생 연구)
            - 논문제출기한 = 수료년도(AJU\_MST.FNSH\_YMD) + 수료이후연수(AJG\_THSS\_SBMSN\_TERM.FNSH\_AFTR\_YCNT)

    - 재학생
        - 대학의 논문신청기준에서 연구윤리이수여부(AJG\_THSS\_APLY\_CRTR.RSCETH\_CPL\_YN)가 체크된 경우, 연구윤리이수여부가 있어야 함

<br>

- 학위청구논문신청 조건
    - 공통(수료생, 재학생)
        - 종합시험통과여부
    - 수료생
        - 논문제출기한
    - 재학생
        - 연구윤리이수여부
    - 체크하는지 불분명
        - 외국어시험통과여부

<br>

- 종합시험통과여부 → 체크하고 있음

```java
CASE WHEN (SELECT JGMT FROM TABLE(DREAMY.F_AJU_GET_GSCHL_GRA_ASESS_RSLT_TBL('', A.STUNO)) WHERE SE_TTL = '외국어시험') = '불합격'
                        THEN 'N' ELSE 'Y' END AS FRGNL_EXAM_PASG_YN             /*외국어 시험 통과 여부YN*/
```

- 외국어시험통과여부

```java
CASE WHEN (SELECT JGMT FROM TABLE(DREAMY.F_AJU_GET_GSCHL_GRA_ASESS_RSLT_TBL('', A.STUNO)) WHERE SE_TTL = '종합시험') = '불합격'
                        THEN 'N' ELSE 'Y' END AS OAEXAM_PASG_YN                 /*종합시험 통과 여부YN*/
```

- 연구윤리이수여부 → 학위청구논문신청기준

```java
CASE WHEN H.RSCETH_CRD > 0 THEN 'Y' ELSE 'N' END   AS  RSCETH_CPL_YN        /*연구윤리 이수여부*/
```

    - 성적마스터의 연구윤리이수과목이 있나
