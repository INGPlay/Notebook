```java
2. 데이터 통계 관련
 (1)전체 통계
  - 전체 누적 마일리지 총합
  - 평균 1인당 누적 마일리지
  - 등록된 전체 활동 건수(예: 빌드업 00건, 스킬업 00건)

 (2)학생별 통계 - ROW
  - 학생 누적 마일리지 순위(내림차순)
  - 개인별 누적 마일리지 변화 그래프(월별)_학생검색 - 원차트 RD
  - 활동 유형별 적립 내역(자격증, 인턴십 등)_학생검색
  - 학과/학년별 총/평균 마일리지

 (3) 활동 유형별 통계
  - 활동 유형별 누적 마일리지 현황
  - 활동 유형별 평균 마일리지 - 원차트 RD
  - 구성 항목 간 점유율(자격증 00%, 어학 00% 등)

 (4) 우수자 선발/장학금 지급 현황
  - 장학금/포상 수혜자 평균 마일리지(종합 / 빌드업 / 스킬업 구분)
  - 장학금/포상 수혜자 학과 / 학년 / 졸업 / 지역청년 비율

  (5) 각 항목별 엑셀 다운로드
```
