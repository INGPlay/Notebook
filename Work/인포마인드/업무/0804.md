    - 2차 인증 퍼블 완료

\*

```java
실장님 확인사항으로 관련 부분 요청합니다.
1. 캠퍼스맵 포틀릿 클릭시 바로 이동하면 좋겠다고 합니다.

2. 모바일 상단 '당직정보'가 아래 형태로 나왔으면 좋겠가도 합니다.
당직정보
(상황실)

3. 당직정보 포틀릿 클릭시 상세가 '* 당직실(754-3000)/종합상황실(754-2558~9)'에서 수정되면 좋겠다고 합니다.
* 종합상황실(754-2558~9)(전화아이콘)
* 당직실(754-3000)(전화아이콘)
그리고 754-2558 같은 곳에 링크처럼 언더바가 생기면 좋겠다고 합니다.(누르면 전화가 갈거라고 느낄 수 있을 것 같아 요청함)

4. 분명 6139에서 열이 처리 되었다고 했었으나 처리가 안 된 것 같습니다.(첨부파일 요청화면2)

추가 요청사항 있으시면 이야기 해 주시고, 실장님이 언제까지 처리될지도 알고 싶어 하셔서 처리 가능일자 미리 연락 주시면 좋겠고, 처리되면 다시 연락바랍니다.
(대략 1주 정도 지나면 확인 요청 하시는 것 같습니다.)
```
