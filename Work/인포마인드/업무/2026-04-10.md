### **1\. 파일 구분 및 저장 방식**

- **BLOB 필요성**: BLOB 저장 옵션 추가 검토
- **임시 파일 관리**:
    - temp 폴더 → 임시 저장
    - Prefix + 연월일로 구분 → 영구 저장
    - 배치 작업으로 정기적 삭제
- **테이블 설계**: 파일 저장 방식에 따라 JOIN 구조 차별화
    - 파일 테이블
    - BLOB 테이블

### **2\. 보안**

- **API Key 관리**: 좀 더 고민
- **인증 프로세스**:
    - ID/PW 로그인 → 2차 인증 (옵션) → 본인인증 → 메시지 매칭 후 로그인 완료
    - 2차 인증 후 비밀번호 변경 가능
- **비밀번호 정책**: 별도 정책 수립 필요
    - 변경일자 표시
    - 만료일자 표시
    - 비밀번호 변경일자 (배치로 자동 입력)
