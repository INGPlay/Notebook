        - 취득예정학점 및 합계

    - [x] 연구윤리준수확인서

    - [x] 연구윤리이수확인서
    - [x] 논문발표실적 : 박사과정만

- [x] 연구윤리제출여부 Y 인 경우, 연구윤리준수확인서 리포트 출력

    - 기존 테이블에 있던 컬럼이 아니라, 연구윤리결과관리 테이블을 바라보게 함

- [x] 논문발표실적 - 논문실적관리(BGP6050209) 의 연구실적리스트를 팝업 띄워서 보여주기
    - [x] 엑셀 버튼 있어야 함
- [x] 출력물에 서명 다 빼기
- [x] 부실학회지 막기→ 부실학회지 팝업 버튼, 통합 서류에서 보이는 것도 막음
- [x] 지도교수 - 지도학생만 보이도록→ 조건은 줬는데, 권한은 아직 제한하지 않음
- [x] 지도교수만 권한 주기→ 모든 교수에게 권한을 줌
